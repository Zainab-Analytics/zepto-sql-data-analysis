# 🛒 Zepto SQL Data Analysis

## 📌 Project Overview

This project is an end-to-end SQL Data Analysis project based on Zepto's inventory dataset.

The objective is to analyze product inventory, pricing, discounts, stock availability, and business performance using SQL.

---

## 📊 Dataset Information

The dataset contains information about:

* Product Category
* Product Name
* MRP (Maximum Retail Price)
* Discount Percentage
* Available Quantity
* Discounted Selling Price
* Product Weight
* Out of Stock Status

Total Records: **3,732**

## 🔍 Business Problems Solved

* Top products with highest discounts
* Most expensive products
* Category-wise inventory analysis
* Out-of-stock product analysis
* Revenue estimation by category
* Price-per-gram analysis
* Weight category segmentation

---

## 📈 Key Insights

* Fruits & Vegetables contain a large share of inventory.
* Discount percentages generally range between 10% and 20%.
* Several high-value products are frequently out of stock.
* Price-per-gram analysis helps identify value-for-money products.
